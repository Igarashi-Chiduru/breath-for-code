# breath for code

##

version: -6.0

##### 安装依赖

```bash
$ yarn  # npm i 获取会导致运行失败，请使用yarn，或改theme源码
```

##### 启动服务

```bash
$ npm run start
```

**编译文档请执行如下命令**

```bash
$ cd /note
$ npm run docs:build
```

###### Build library via `father-build`,

```bash
$ npm run build
```
